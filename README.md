# Local Service Business Website Redesign

## Project Overview
This project focuses on redesigning a local beauty salon website with a strong emphasis on lead generation, user experience, and conversion optimization.

## Pages Designed
1. Homepage
2. Services Page
3. Contact / Appointment Page

## Features
- Modern UI Design
- Mobile-Friendly Layout
- Strong Call-To-Action Buttons
- Service Showcase Section
- Contact & Appointment Form
- Customer Testimonials

## Tools Used
- Canva

## Design Goal
To improve user engagement, increase appointments, and provide a clean user experience.
